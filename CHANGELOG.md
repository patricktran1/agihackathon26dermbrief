# Changelog

All notable changes to DermBrief EvidenceOps will be documented in this file.

The project follows semantic versioning for tagged releases.

## [Unreleased]

### Added

- Node 22 native test suites for journal normalization, evidence ranking, and AI governance.
- Native test coverage collection in continuous integration.
- Explicit syntax, test, type-check, and production-build gates.
- CodeQL scanning and automated dependency update configuration.
- Contribution, security, conduct, ownership, issue, and pull request guidance.
- Architecture decision records for safety-critical design choices.

### Changed

- Expanded CI from an implicit build-time regression path to visible, independently named quality checks.

## [1.4.0] - 2026-07-19

### Added

- Today’s DermBrief cross-journal evidence inbox.
- Governed LLM Grounder and separate Safety Auditor.
- Deterministic AI vetoes and fallback behavior.
- Governed AI architecture and submission documentation.

## [1.0.0] - 2026-07-18

### Added

- Five-stage EvidenceOps workflow.
- PubMed retrieval and curated-journal validation.
- Deterministic evidence appraisal.
- Claim-level evidence mapping.
- Physician-gated Publisher stage.
